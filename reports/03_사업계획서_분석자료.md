## 2. 제품 구성

### 2-1. 웹 포탈 (대리점용) — 24개 페이지
- 대시보드 (KPI, 실시간 현황)
- 기사 관리 (등록, 문서, 프로필)
- 계약서 관리 (21종 템플릿, 전자서명, 변경이력)
- 정산 관리 (엑셀 업로드, 5종 PDF, 세금계산서)
- 매출 리포트
- 공지사항 관리
- 설정 (프로필, 결제, 관리자, 도장, 알림)

### 2-2. 관리자 포탈 (플랫폼 운영) — 13개 페이지
- 구독사 관리 + 플랜 변경
- 구독/결제 현황 (MRR, 연체)
- 시스템 공지 관리
- 감사 로그
- 서버 상태 모니터링
- 플랜 설정 (동적 편집)

### 2-3. 모바일 앱 (기사용) — 19개 화면
- 정산 확인 + 푸시알림
- 전자계약 서명
- 교육 수료
- 문서 제출

### 2-4. API 시스템 — 24개 엔드포인트
- RESTful API 아키텍처
- Zod 스키마 검증
- CSRF/Rate Limiting 보호

## 6. 시장 분석

### 6-1. TAM (Total Addressable Market)
- 국내 택배 대리점/영업소: 약 **15,000개소** (CJ, 한진, 롯데, 로젠, 우체국 등)
- 퀵서비스/배달 대리점: 약 **5,000개소**
- **TAM ≈ 20,000개소 x ₩99,000/월 = ₩237억/년**

### 6-2. SAM (Serviceable Available Market)
- 디지털화 의향이 있는 중대형 대리점: 약 **5,000개소**
- **SAM ≈ 5,000 x ₩99,000/월 = ₩59억/년**

### 6-3. SOM (Serviceable Obtainable Market) — 3년 목표
- 1년차: 100개소 (점유율 2%)
- 2년차: 300개소 (점유율 6%)
- 3년차: 500개소 (점유율 10%)
- **3년차 SOM ≈ ₩57억/년**

### 6-4. 경쟁 환경
| 경쟁사/대안 | 특징 | 약점 |
|------------|------|------|
| 엑셀 수작업 | 현재 90%+ 사용 | 오류, 시간 소요, 감사 불가 |
| 더존/영림원 ERP | 대기업용 | 택배 특화 X, 고가 |
| 직접 개발 | 대형 물류사 | 소규모 대리점 불가 |
| **logiSSign** | **택배 특화 SaaS** | **시장 진입 필요** |

## 7. 핵심 경쟁력 (차별화 포인트)

### 7-1. 정산 자동화
- 엑셀 업로드 → 자동 컬럼 분류 → 수식 계산 → PDF 5종 자동 생성
- 기존: 대리점당 **월 3~5일** 수작업 → logiSSign: **1시간 이내**

### 7-2. 전자계약 내장
- 21종 표준 계약서 템플릿 (업계 최초 내장형)
- QR 코드 진위확인
- 법적 효력 있는 전자서명

### 7-3. 모바일 기사 앱
- 기사가 직접 정산 확인 + 계약 서명
- 푸시 알림으로 실시간 소통
- 대리점-기사 간 분쟁 감소

### 7-4. 플랜 기반 SaaS
- 무료 시작 → 유료 전환 (프리미엄 모델)
- 관리자가 동적으로 플랜/기능 설정 가능
- 사이드바 메뉴 잠금 → 자연스러운 업셀
